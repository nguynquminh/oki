## 2024-05-24 - Parallelize I/O bound tasks in Discord Bot\n**Learning:** Sequential `await` in loops over multiple entities (e.g., creating team channels or generating transcripts) adds significant round-trip latency overhead when interacting with the Discord API.\n**Action:** Use `Promise.all()` to parallelize independent asynchronous operations (like creating channels for different teams or fetching messages for multiple channels), allowing `discord.js`'s built-in rate limiter to handle queuing efficiently while minimizing total execution time.
